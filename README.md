# DEPRECATED: gavant-ember-loading-button

**This project has been deprecated. Please use [gavant-ember-button-spinner](https://github.com/Gavant/gavant-ember-button-spinner) instead.**

A button component that shows a spinner while waiting for the click action to resolve.

**DISCLAIMER: This addon is not actively maintained for public use. Pull requests are welcome, but we do not guarantee  responses to bug submissions or feature requests, so use at your own risk.**

## Installation

* `git clone` this repository
* `npm install`
* `bower install`

## Running

* `ember server`
* Visit your app at http://localhost:4200.

## Running Tests

* `npm test` (Runs `ember try:testall` to test your addon against multiple Ember versions)
* `ember test`
* `ember test --server`

## Building

* `ember build`

For more information on using ember-cli, visit [http://ember-cli.com/](http://ember-cli.com/).
